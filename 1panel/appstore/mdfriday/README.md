## 产品介绍

**MDFriday** 是一款数字资产创作助手。在 Obsidian 中写作，跨设备同步，即时发布为专业网站。

## 主要功能

- **即时发布**：在 Obsidian 中编写 Markdown，一键发布为专业网站。
- **多设备同步**：通过 CouchDB 实现跨设备实时同步。
- **丰富主题**：500+ 精美主题，涵盖文档、博客、简历、作品集、幻灯片等。

## 使用说明

### 快速开始（IP 访问）

只需填写服务器 IP，1Panel 自动分配端口。安装后直接访问：

```
http://你的服务器IP:端口
```

此时请求直达 Hugoverse，不经过 Caddy。

### 绑定域名（可选）

如有域名，填写域名和 HTTPS 端口后：
1. 在 1Panel 中创建网站，反向代理到 `http://127.0.0.1:HTTP端口`
2. 申请 SSL 证书并启用 HTTPS

### 启用 DNSPod（可选）

如需 Caddy 自动管理 DNS 证书，设置 `DNSPOD_ENABLED=true` 并填写凭据。
